Hysteresis Function

Develop a robot that follows you through the airport and carries your bag

You want the robot to follow closely, but not too closely 1 m distance

if ( distance_to_person > 1 m ):
  move_forward()
elif ( distance_to_person < 1m ):
  move_backward()
else:
  stop()

This algorithm will have the robot oscillate between move_forward and move_backward continuously, because changes that we are at exactly 1m are extremely slim

To solve this problem, we can use a hysteresis function (dead zone): zone where we are not allowed to switch our control (to avoid oscillation)

if ( dist > 1.05 m ):  # high
  cmd = move_forward()
elif ( dist < 0.95 m): # low
  cmd = move_backward()
else: # between high and low
  if ( dist < 1 m ) and ( prev_cmd == move_backward ):
    cmd = move_backward()
  elif ( dist > 1 m ) and ( prev_cmd == move_forward ):
    cmd = move_forward()
  else:
    stop() 
prev_cmd = cmd

if Sobel(pixel) > high:
  pixel = EDGE
elif Sobel(pixel) < low:
  pixel = NON_EDGE
else:
  if Neighbor(Pixel) == Edge:
    pixel = EDGE 
  else:

Non-maxima suppression: suppress neighbours that have non-maxima Sobel response to avoid multiple edge pixels for an edge (common problem of first order operators)
